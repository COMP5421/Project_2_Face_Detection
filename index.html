COMP5421 / Project 2 / Face Detection

Example of face detection.

In this project we implement a face detection program using HoG (vl_hog) based on Triggs's paper.

The whole program mainly contains these steps.

  1. Extract Histogram of Oriented Gradient(HOG) features from positive samples.
  2. Extract Histogram of Oriented Gradient(HOG) features from random negative samples.
  3. Train a Linear SVM classifer from both positive and negative samples.
  4. Detact test dataset by sliding window and adjusting the scale of window(multiscale), determing whether each window contains a face or not.
  5. Generate a bounding box with confidence threshold.
  6. Compute ROC, precision-recall curve and average precision.

We have trained our linear SVM with 6713 cropped images containing positive features from Caltech Web Faces project and 10000 images containing random negative from SUN database.

Train linear SVM

We use Linear SVM (vl_svmtrain) with lambda as 0.0001 to get a linear classifier.

Multi-scaling and step size

We have prepared several scales to resize the original photo and 3 steps to detect the human face.
